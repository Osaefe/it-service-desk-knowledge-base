# Ticket 002: Printer Mapping Failure

## Summary
User unable to print to network printer.

## Symptoms
- Printer offline in Windows
- Error: “Cannot connect to printer”

## Investigation
- Verified printer is powered and connected to network  
- Checked printer driver installation  
- Pinging printer IP successful  

## Root Cause
Outdated printer driver  

## Resolution
- Updated printer driver  
- Mapped printer manually  
- Verified successful print job  

## Outcome
Printer functionality restored; documented for future reference.
