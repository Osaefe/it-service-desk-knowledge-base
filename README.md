# IT Service Desk Knowledge Base

This repository simulates a professional IT service desk environment, demonstrating ticket management, end-user support, and structured troubleshooting procedures.

## Project Overview
The project documents common IT support issues encountered in corporate environments. Each “ticket” represents a realistic incident with step-by-step investigation, resolution, and documentation.

## Support Scenarios Simulated
1. Wi-Fi connectivity failure  
2. Printer mapping and driver issues  
3. Password reset and account lockout  
4. Performance degradation on Windows PCs  
5. Software installation and update failures  

## Technical Scope
- End-user support for Windows operating systems  
- Network connectivity troubleshooting  
- Printer and peripheral configuration  
- Password and account management  
- Software deployment and troubleshooting  

## Skills Demonstrated
- Helpdesk workflow management  
- Ticket documentation and resolution  
- Root cause analysis and problem-solving  
- User communication and technical guidance  
- Windows diagnostics and troubleshooting  

## Tools Used
- Windows 10 / 11  
- Active Directory  
- Office 365 / Microsoft Teams  
- Remote Desktop  
- Command-line troubleshooting (cmd, PowerShell)  

## Outcome
Provides a complete portfolio of IT service desk scenarios, demonstrating practical skills relevant to:  
- IT Support Analyst  
- Desktop Support Technician  
- Helpdesk Associate  
- Junior Systems Administrator
